
# Telegram Channel Analyzer

## Быстрый старт

1. Заполните `.env.example`, затем переименуйте его в `.env` (ключи API, URL базы и т.д.).

2. Установите зависимости:

   ```bash
   py -3 -m pip install -r requirements.txt
   ```

3. Запустите приложение:

   ```bash
   py -m app.main
   ```
4. Авторизироваться - номер телефона и код подтверждения

5. Добавиться в необходимые группы.

6. Пользователи, которых нужно добавить, должно прислать хотя бы одно сообщение.
